## Getting Started

To get started with Issue Tracker, follow these steps:

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/beiryu/issue-tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the web server
   ```sh
   npm run dev
   ```
